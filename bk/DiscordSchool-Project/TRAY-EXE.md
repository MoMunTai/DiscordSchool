# EXE — รันบอทแบบหน้าต่าง console

## ไฟล์ผลลัพธ์ (`dist/`)
- **DiscordSchool.exe** — ตัวโปรแกรม (Node + บอท + config ฝังในตัวครบ ไฟล์เดียวจบ)

## วิธีใช้
1. **ดับเบิลคลิก `DiscordSchool.exe`**
2. รอบแรกจะขึ้น `Preparing first run (extracting files), please wait...` (แตกไฟล์ ~20-40 วิ ครั้งเดียว รอบต่อไปเร็ว)
3. หน้าต่าง console จะโชว์สถานะตามจริง:
   ```
   📡 Discord School Broadcast
   🚀 เริ่มระบบ...
   👂 LeaderBot กำลังฟังในห้อง: Leader Room
   🔊 A1Bot พร้อมเล่นเสียงในห้อง: A1
   ...
   ✅ กำลังทำงาน — บอททั้งหมดออนไลน์แล้ว
   ❌ ปิดโปรแกรม: กดปุ่ม X มุมขวาบนของหน้าต่างนี้
   ```
4. **ปิดโปรแกรม:** กดปุ่ม **X มุมขวาบน** ของหน้าต่าง (หรือ Ctrl+C) — บอทจะตัดการเชื่อมต่อทั้งหมด

## กันเปิดซ้ำ
ถ้าเผลอเปิด exe ซ้ำตอนที่ตัวแรกยังรัน → หน้าต่างที่ 2 จะขึ้น `⚠️ โปรแกรมเปิดอยู่แล้ว` แล้วปิดเอง (กัน token ชน)

## เปิดอัตโนมัติตอน Windows บูต (ออปชัน)
1. `Win + R` → พิมพ์ `shell:startup` → Enter
2. ก๊อป **shortcut ของ DiscordSchool.exe** ไปวาง

## config ฝังในตัว exe
- token / ห้อง / role ถูกฝังตอน build → exe ไฟล์เดียวรันได้เลย ไม่ต้องมีไฟล์ข้างนอก
- **เปลี่ยน token/ห้อง/role** → แก้ `config.js` / `.env` ในโปรเจกต์ → `npm run build` ใหม่
- ปรับ volume รายห้องสด: `/setvoltb`, ทุกห้อง: `/setvolall`, ดู: `/voltb` (ไม่ต้อง build ใหม่)

## build
```
npm run build       → dist/DiscordSchool.exe (config ฝังในตัว ~56 MB)
```
โหมด dev (รันตรงจากโปรเจกต์ ใช้ config.js/.env ในโปรเจกต์):
```
npm start           → รัน relay ตรงๆ
```

## ย้ายไปเครื่องอื่น
- ก๊อปแค่ **DiscordSchool.exe** ไปเครื่อง **Windows 64-bit** ได้เลย (ไม่ต้องมี config/Node)
- กฎเหล็ก: **รันได้ที่เดียวเท่านั้น** (token ชนถ้ารัน 2 เครื่อง — มีตัวกันเปิดซ้ำในเครื่องเดียวกันแล้ว)
